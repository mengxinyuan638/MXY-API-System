# 萌新源API+API管理后台

#### 介绍
基于layui以及pear-Admin-layui开发的API管理后台

#### 版本要求
PHP5.0以上版本
Mysql



#### 安装教程
将源码上传至网站根目录，然后登录网站数据库，新建一张user表单并在表单内添加username,password
默认api管理后台路径网址/admin-mxy
注意，到手后请将默认的admin文件夹修改为你个人路径，因为现在还没有做安全防护处理，所以请务必修改默认路径
